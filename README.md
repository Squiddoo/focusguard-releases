# FocusGuard Releases

Here's where you download **FocusGuard** — a lightweight Windows app that blocks distracting apps until you say they're allowed again.

## Download

Get the latest version:
**[FocusGuard-Setup-1.0.8.exe](https://github.com/Squiddoo/focusguard-releases/releases/download/v1.0.8/FocusGuard-Setup-1.0.8.exe)**

Windows 10/11 (x64) • ~90 MB installer

## What It Does

Lock apps you waste time on, set an unlock time, and they don't open until that time comes. Pretty simple.

- Lock multiple apps at once
- Add a reason (helps you remember why you locked it)
- If you mess up and lock yourself too long, emergency unlock gives you 15 min
- Settings to control notifications, fullscreen, auto-updates
- Updates happen silently in the background

## Install & Run

1. Download the installer
2. Run it
3. Open FocusGuard
4. Pick an app, set when it unlocks
5. Done

## Features

- **Lock multiple apps** with different unlock times
- **Auto-updates** download and install silently (like Discord)
- **Settings drawer** to customize behavior
- **Emergency unlock** if you need out (15 min pause)
- **Tray menu** to minimize/quit quickly
- **About panel** shows version info

## How Updates Work

Updates check automatically when you start the app. If there's a new version:
- Downloads in background
- Shows a progress bar at the top
- Restarts automatically
- Your locks and settings stay safe

## Release History

See all versions: [Releases](https://github.com/Squiddoo/focusguard-releases/releases)

## Known Limitations

- Can't lock system critical apps (Windows protects those)
- Someone with admin access could force things open
- That's intentional — this is for self-discipline, not security
